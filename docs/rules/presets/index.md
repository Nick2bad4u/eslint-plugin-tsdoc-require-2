# Presets Overview

Use presets to roll out `eslint-plugin-tsdoc-require-2` in stages, from minimal
enforcement to strict documentation coverage.

## Preset catalog

- [recommended](./recommended.md)
- [detailed](./detailed.md)
- [packages](./packages.md)
- [typedoc](./typedoc.md)
- [typedoc-strict](./typedoc-strict.md)
- [tsdoc](./tsdoc.md)
- [jsdoc](./jsdoc.md)
- [all](./all.md)
