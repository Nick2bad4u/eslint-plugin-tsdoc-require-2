# typedoc

Conservative TypeDoc-oriented preset with declaration-kind tags that are safe to infer.

## Config key

`tsdocRequire.configs.typedoc`

## Flat Config example

```ts
import tsdocRequire from "eslint-plugin-tsdoc-require-2";

export default [
    tsdocRequire.configs.typedoc,
];
```
