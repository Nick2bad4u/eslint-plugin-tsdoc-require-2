# recommended

Baseline preset that only enforces TSDoc comment presence via `tsdoc-require-2/require`.

## Config key

`tsdocRequire.configs.recommended`

## Flat Config example

```ts
import tsdocRequire from "eslint-plugin-tsdoc-require-2";

export default [
    tsdocRequire.configs.recommended,
];
```
