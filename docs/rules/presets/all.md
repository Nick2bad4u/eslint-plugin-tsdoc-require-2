# all

Enables every rule provided by this plugin, including all require-tag rules and `restrict-tags`.

## Config key

`tsdocRequire.configs.all`

## Flat Config example

```ts
import tsdocRequire from "eslint-plugin-tsdoc-require-2";

export default [
    tsdocRequire.configs.all,
];
```
